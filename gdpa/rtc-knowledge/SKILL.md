# RTC 实时音视频平台

RTC（实时音视频）平台提供高质量、低延迟的实时音视频通信能力，支持多平台 SDK 接入和丰富的服务端 OpenAPI，适用于音视频通话、互动直播、在线教育等多种场景。

## SDK 接入

RTC 服务端接入主要通过 OpenAPI 调用实现，支持房间管理、转推直播、状态监控等核心功能。接入前需要在控制台开通服务并获取 AK/SK 凭证。

**主要使用场景**：
- **房间管理**：解散房间、移出用户、获取房间信息等操作
- **转推直播**：将房间内音视频流合流转推到 CDN 进行直播分发
- **消息回调**：接收用户进出房间、流状态变化等实时事件通知
- **环境配置**：支持 BOE 测试环境和正式环境的接入配置
